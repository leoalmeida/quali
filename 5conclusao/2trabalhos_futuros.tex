\section{Trabalhos Futuros}
Os próximos passos desse projeto seria a instalação fisica do protótipo em um piloto na
cidade de São Carlos e coletar dados para realizar testes e análises de regras por meio de gráficos. Este piloto serve também como importante teste final e para realizar possíveis modificações no protótipo.

Em uma linha de trabalho diferente, o protótipo desenvolvido seria integrado a serviços
web, de forma disponibilizar os dados de maneira .
